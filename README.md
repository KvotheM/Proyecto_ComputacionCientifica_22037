modelado_epidemico.ipynb contiene el código que simula el comportamiento de los diferentes agentes virales y genera gráficas que facilitan la interpretación del comportamiento
de los sistemas junto con el efecto que estos tienen en el sistema de salud. 


Gráficas_AgentesVirales.ipynb contiene código para generar y almacenar de manera automática las gráficas de la simulación.
